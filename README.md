# BI RH/DP — Nosso Atacarejo

Protótipo visual de 9 dashboards Power BI para o setor de RH/DP.

## Dashboards
1. Colaboradores
2. Controle de Férias
3. Controle de Horas (placeholder)
4. Folha de Pagamento
5. Empréstimos
6. Rescisões
7. Tempo de Empresa
8. Afastamentos
9. Atestados

## Como rodar
```bash
npm install
npm run dev
```

## Scripts SQL
Os scripts completos estão na pasta `scripts-sql/`.
Cada dashboard tem um botão **📜 Script** que exibe o SQL da consulta.

## Stack
- React 18 + Vite
- CSS puro (sem libs de charts)
- Dados fictícios para prototipagem
